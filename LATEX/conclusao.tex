\section{CONCLUSÃO \label{conclusao}}

\hspace*{1.25cm}Neste trabalho foram apresentadas técnicas de
detecção de transição baseadas em diversas metodologias e para cada
tipo de transição foi adotada uma metodologia específica. Foi
observado durante os testes de detecção de cortes que dependendo das
imagens presentes no vídeo, é necessário adequar o filtro de Canny
para que seja feita uma detecção adequada, visto que os resultados
obtidos a partir da aplicação deste filtro nas imagens variam
conforme o comportamento da luminância dos frames em análise. Os
valores aqui utilizados como padrão ficam vulneráveis à transições
entre duas tomadas que sejam muito escuras, porém se estes limiares
forem modificados para detectar estas transições, as transições
entre tomadas mais claras podem deixar de ser detectadas, além de
aumentar o número de falsos positivos.
