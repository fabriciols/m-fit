\chapter*{\centering RESUMO \label{resumo}}

\hspace*{1.25cm}Na área de Editoração de Vídeos, uma das atividades
mais demandadas é a detecção de transição de tomadas, que é
basicamente a concatenação do final de uma tomada com o inicio de
outra. Essa tarefa tem várias utilidades em diversas aplicações,
como indexação, recuperação, análise e edição de vídeos. Estas
transições podem ocorrer de diversas formas, tais como: cortes,
\textit(fades) ou \textit(dissolve). Este trabalho apresenta o
M-FIT, um framework que possibilitará o reconhecimento automático
das transições de tomadas de cenas a partir de uma determinada
amostragem de imagens (frames de um vídeo) através do uso de
técnicas de Ritmo Visual e Segmentação de Imagens. Além de facilitar
o processo de detecção de imagens, o M-FIT permite ao usuário
aplicar efeitos básicos de edição de imagens no vídeo, de forma
rápida e simples. Sua interface é baseada em QT, o que torna o M-FIT
um sistema com uma interface simples e de fácil entendimento.
