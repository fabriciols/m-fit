\subsection{Carregar um Vídeo\label{carrega_video}}

\hspace{1.25cm}Nesta seção detalharemos os passos necessãrios para se carregar um Vídeo.

\begin{enumerate}

\item{Abra o MFIT (Figura \ref{img:mfit_aberto})}

\begin{figure}[h|top]
 \centering
 \includegraphics[width=0.7\linewidth]{imagens/mfit_aberto.png}
 \caption{MFIT Aberto sem Vídeo carregado.}
 Fonte: Autor.
 \label{img:mfit_aberto}
\end{figure}

\item{Clique com o botão direito no menu "Arquivo", opção "Carregar Vídeo".
Se preferir, pressione as teclas Ctrl+Shif+O (Figura \ref{img:carregar2}). }

\begin{figure}[h|top]
 \centering
 \includegraphics[width=0.7\linewidth]{imagens/carregar2.png}
 \caption{Menu "Arquivo", opção "Carregar Vídeo" .}
 Fonte: Autor.
 \label{img:carregar2}
\end{figure}

\item{Na janela que abrirá, selecione um Vídeo válido (com extensão .AVI) e de um duplo clique (Figura \ref{img:carregar3}). }

\begin{figure}[h|top]
 \centering
 \includegraphics[width=0.7\linewidth]{imagens/carregar3.png}
 \caption{Selecionando um Vídeo válido.}
 Fonte: Autor.
 \label{img:carregar3}
\end{figure}

Note que uma janela se abrirá, questionando se o processo de detecção de transições deve ser iniciado.
Por hora, clique em cancelar (Figura \ref{img:carregar4}). }

\begin{figure}[h|top]

\item{Processo finalizado, o Vídeo está pronto para se efetuar todas as operações possíveis (Figura \ref{img:carregar5}). }

\begin{figure}[h|top]
 \centering
 \includegraphics[width=0.7\linewidth]{imagens/carregar5.png}

\subsection{Detecção de transições\label{carrega_video}}

\hspace{1.25cm}Nesta seção detalharemos os passos necessãrios para se detectar as transições de um Vídeo.

\hspace{1.25cm}Para esta funcionalidade temos dois caminhos:

\begin{enumerate}
\item{À partir do carregamento do Vídeo}

\subitem{ Ao chegar no passo 4 do processo de carregamento de Vídeo (Seção \ref{carrega_video}), existe a opção de aceitar a detecção de todas as transições.
Para isso, basta clicar em "OK" quando o sistema quiestionar sobre a detecção. Com isso o processo de detecção de todas as transições se iniciará. ( Figura \ref{img:carregar4} )}

\item{Após o carregamento do Vídeo}

\subitem{ Após realizar todos os passos do processo de carregamento do Vídeo (Seção \ref{carrega_video}), 
os botões de detecção de transição estarão habilitados. Escolha a detecção que deseja e pressione o botão (Figura \ref{img:carregar5} )}

\item{ Após escolher um dos passos acima, o sistema realizará a detecção de transições, com isso as transições ficarão delimitadas
na Timeline, e a lista de transições será prenchida (Figura \ref{img:transicoes1}).}

\end{enumerate}

\begin{figure}[h|top]
 \centering
 \includegraphics[width=0.8\linewidth]{imagens/transicoes1.png}
 \caption{Transições Detectadas.}
 Fonte: Autor.
 \label{img:transicoes1}
\end{figure}

\begin{figure}[h|top]
