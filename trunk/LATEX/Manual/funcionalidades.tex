\section{FUNCIONALIDADES DO MFIT\label{introducao}}

\hspace*{1.25cm}Neste capítulo será explicado todas as funcionalidades do MFIT.

\subsection{Carregar um Vídeo\label{carrega_video}}

Nesta seção detalharemos os passos necessãrios para se carregar um Vídeo.

\begin{enumerate}

\item{Abra o MFIT (Figura \ref{img:mfit_aberto})}

\begin{figure}[h|top]
 \centering
 \includegraphics[width=0.5\linewidth]{imagens/mfit_aberto.png}
 \caption{MFIT Aberto sem Vídeo carregado.}
 Fonte: Autor.
 \label{img:mfit_aberto}
\end{figure}

\item{Clique com o botão direito no menu "Arquivo", opção "Carregar Vídeo".
Se preferir, pressione as teclas Ctrl+Shif+O (Figura \ref{img:carregar2}). }

\begin{figure}[h|top]
 \centering
 \includegraphics[width=0.5\linewidth]{imagens/carregar2.png}
 \caption{Menu "Arquivo", opção "Carregar Vídeo" .}
 Fonte: Autor.
 \label{img:carregar2}
\end{figure}

\item{Na janela que abrirá, selecione um Vídeo válido (com extensão .AVI) e de um duplo clique (Figura \ref{img:carregar3}). }

\begin{figure}[h|top]
 \centering
 \includegraphics[width=0.5\linewidth]{imagens/carregar3.png}
 \caption{Selecionando um Vídeo válido.}
 Fonte: Autor.
 \label{img:carregar3}
\end{figure}

\item{Após alguns instantes o vídeo estará aberto, e sua timeline estará montada.
Note que uma janela se abrirá, questionando se o processo de detecção de transições deve ser iniciado.
Por hora, clique em cancelar (Figura \ref{img:carregar4}). }

\begin{figure}[h|top]
 \centering
 \includegraphics[width=0.5\linewidth]{imagens/carregar4.png}
 \caption{Vídeo carregado e janela questionando o processo de detecção.}
 Fonte: Autor.
 \label{img:carregar4}
\end{figure}

\item{Processo finalizado, o Vídeo está pronto para se efetuar todas as operações possíveis (Figura \ref{img:carregar5}). }

\begin{figure}[h|top]
 \centering
 \includegraphics[width=0.5\linewidth]{imagens/carregar5.png}
 \caption{Fim do processo.}
 Fonte: Autor.
 \label{img:carregar5}
\end{figure}

\end{enumerate}
