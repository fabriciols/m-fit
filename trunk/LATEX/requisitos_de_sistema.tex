\section{Requisitos de Sistema \label{RQS}}

\subsection{Localização de arquivo \label{RQS01}}

O sistema deverá ser capaz de abrir arquivos que estejam em qualquer
tipo de disco de armazenamento, pen drives, HD, CD´s, entre outros.

\subsection{Montagem do Ritmo Visual \label{RQS02}}

Aplicação de cálculos para extração da diagonal principal de todos
os frames do vídeo e transformação em uma coluna do ritmo visual.

\subsection{Montagem de mapa de bordas \label{RQS03}}

Aplicar um filtro passa-baixa, eliminando os ruídos do Ritmo Visual.
Com isso o sistema obterá o mapa de bordas do Ritmo Visual.

\subsection{Tipos de vídeo \label{RQS04}}

O sistema deverá ser compatível no mínimo com vídeos do tipo AVI e
MPEG.

\subsection{Sistema Operacional \label{RQS05}}

O sistema deverá ser desenvolvido para rodar na plataforma Windows,
porém deve-se visar a portalidade do sistema.

\subsection{Digitalização do vídeo \label{RQS06}}

O vídeo fornecido ao sistema deverá ter sido anteriormente
digitalizado pelo usuário, não cabendo ao sistema esta função.

\subsection{OpenCV \label{RQS07}}

Necessária a inclusão da biblioteca OpenCV para que o sistema possa
utilizar funções de PDI.

\subsection{QT \label{RQS08}}

A interface do sistema deverá ser desenvolvida através da ferramenta
QT.
