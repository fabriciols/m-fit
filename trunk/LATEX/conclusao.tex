\section{CONCLUSÃO \label{conclusao}}

\hspace*{1.25cm}Neste trabalho foram apresentadas técnicas de
detecção de transição baseadas em diversas metodologias e para cada
tipo de transição foi adotada uma metodologia específica. Foi
observado durante os testes de detecção de cortes que para cada
imagem a ser filtrada pelo filtro de Canny são necessárias
configurações diferentes quanto ao valor de seus limiares. Os
valores utilizados como padrão ficam vulneráveis à transições entre
duas tomadas que sejam muito escuras, porém se estes limiares forem
modificados para detectar estas transições, as transições entre
transições mais claras deixarão de ser detectadas, visto que tomadas
mais claras são mais comuns na maioria dos vídeos, optou-se por
detectar transições entre este tipo de tomada para que o sistema
obtivesse um melhor desempenho.
